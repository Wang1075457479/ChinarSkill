# Trello任务创建工具

## 功能特点
1. 自动标准化标题格式：【分类】-任务名称
2. 自动生成标准化清单，统一使用1、2、3序号
3. 支持XMind文件导入，每个画布生成一个任务卡片
4. 自动整理看板任务，批量标准化格式
5. 自动分配成员、计算截止日期

## 配置方法
1. 打开 `references/config.example.json`
2. 填写你的Trello API Key、Token、看板ID
3. 另存为 `references/config.json`

## 使用方法
### 1. 快速创建任务
```bash
cd scripts
python create_task.py <分配人> <任务标题> [附件链接]
```

### 2. 导入XMind任务
```bash
cd scripts
python import_xmind_task.py <xmind文件路径> [分配人]
```

### 3. 自动整理看板
```bash
cd scripts
python trello_auto_organizer.py
```

## 依赖安装
### 方法1：双击运行 `安装依赖.bat` (推荐Windows用户使用)
### 方法2：手动执行命令
```bash
pip install requests xmindparser
```

## API Key获取
1. 访问 https://trello.com/app-key 获取API Key
2. 点击"Token"链接生成访问令牌
