# 任务创建示例

## 分类说明

支持的任务分类（≤5字）：
- **开发** - 新功能开发
- **优化** - 性能/体验优化
- **安全** - 安全相关
- **规则** - 规范/流程制定
- **设计** - 设计相关
- **测试** - 测试相关
- **文档** - 文档编写
- **重构** - 代码重构

---

## 示例1：简单任务 - 开发类

**用户输入：**
```
创建任务：开发登录页面
```

**拆解结果：**

**标题：** `【开发】-登录页面`

**描述：**
```
开发登录页面
```

**清单：**
```
1、设计登录页面UI
2、实现表单验证
3、对接后端API
4、添加错误提示
5、测试登录功能
```

## 标题生成规则

### 好的标题示例
- ✅ `【开发】-登录页面` - 分类清晰，任务明确
- ✅ `【优化】-性能提升` - 分类+核心任务
- ✅ `【安全】-API鉴权` - 安全类任务
- ✅ `【规则】-提交规范` - 规范制定
- ✅ `【重构】-订单系统` - 重构类任务

### 需要优化的标题
- ❌ `【开发】-开发登录页面功能模块` - 太长（超过15字）
- ❌ `【开发】登录页面` - 缺少短横线分隔
- ❌ `【任务】-登录页面` - 分类不明确
- ❌ `【开发】-做一个东西` - 任务太模糊

### 分类判断规则

| 关键词 | 分类 |
|--------|------|
| 开发、实现、新增、创建、功能 | 开发 |
| 优化、改进、提升、提速、重构 | 优化/重构 |
| 安全、漏洞、鉴权、加密、防护 | 安全 |
| 规范、规则、制度、流程、标准 | 规则 |
| 设计、原型、UI、界面、视觉 | 设计 |
| 测试、用例、验证、回归 | 测试 |
| 文档、手册、指南、说明 | 文档 |

### 标题字数计算

总体不超过15个汉字（含分类、括号、短横线）：
- 分类：2-4字
- 任务标题：5-10字
- 总长度：≤15字
