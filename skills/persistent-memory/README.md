# Persistent Memory - 持久记忆系统

让 OpenClaw 像人类一样拥有长期记忆能力。

## 🎯 解决的问题

- **会话失忆**：每次新会话都忘记之前的对话
- **配置丢失**：重要配置和决策没有记录
- **重复犯错**：同样的错误反复出现
- **上下文断层**：无法维护长期项目的连续性

## 🚀 快速开始

```bash
# 初始化记忆系统
bash ~/.openclaw/skills/persistent-memory/scripts/init.sh
```

## 📁 安装

```bash
cd ~/.openclaw/skills
git clone https://github.com/ChinarG/persistent-memory.git
```
